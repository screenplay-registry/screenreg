The Screenplay Registry Verifier

Drag in your screenplay + manifest + .ots proof. Verification runs entirely in your browser. Nothing is uploaded.

Privacy promise: this page is fully client-side. Your script bytes never leave your browser. Open the Network tab in your dev tools — no upload requests are made. Bitcoin block-header verification (when the proof references one) optionally queries a public block explorer; this version of the verifier reads the asserted block height from the proof and trusts the calendar's claim. For full trustless verification, run the CLI against your own Bitcoin node.
